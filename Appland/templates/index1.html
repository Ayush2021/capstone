What we have for you?

Transform any hand-drawn design into a HTML code with AI. Use are highly user interactive platform to sketch your website, our AI based model will bring the sktech to life!

Detect design patterns

A Custom Vision Model detects meaningful design elements in an image. The model was trained to perform object recognition against HTML hand-drawn patterns.

Understand handwritten text

Each detected element is passed through a Text Recognition Service to extract handwritten content.

Understand structure

Information about the detected objects and the objects’ position inside the image is fed into an algorithm that generates the underlying structure.

Build HTML

A valid HTML is generated according to the detected layout of the design elements.

Designers can collaborate and draw on a whiteboard—then see the resulting code to test their new web page ideas immediately.

CodingSketch uses machine learning to convert hand-written drawings to working HTML prototypes. Designers share ideas on a whiteboard, then changes are shown in the browser instantly.

  • Once a design is drawn, it is usually manually translated into a working HTML wireframe.
  • We can use Computer Vision to build a system that understands what a designer has drawn on a whiteboard, then translates that understanding to HTML code.
  • Custom Vision service trains models to detect HTML objects, then uses text recognition to extract handwritten text in the design. By combining the object and the text, we can generate HTML snippets of different design elements.

We address the challenges that have been plaguing designers and developers for years: that even though designers go to great lengths to create digital solutions that are perfectly measured and consistently crafted, design systems do not fluently translate to development systems.
